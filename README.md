<p align="center">A little API built with NestJs.</p>

## Description

This little API is built around a user crud to give example of common concepts like requests, responses and HTTP methods.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
